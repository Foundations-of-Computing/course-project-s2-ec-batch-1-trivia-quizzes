HTML stands for HyperText Markup Language.

The default value of the "position" property in CSS is "static".

Question 3: What does the 'var' keyword do in JavaScript?

The 'var' keyword is used to declare a variable in JavaScript.

Question 4: What is the purpose of the "for" loop in JavaScript?

A "for" loop is used to execute a block of code repeatedly based on a condition.

Question 5: What is the function of the 'console.log()' method in JavaScript?

An array is a data structure in JavaScript that stores multiple values in a single variable.

"==" compares values, while "===" compares both values and data types in JavaScript.

Question 9: What is a callback function in JavaScript?

A callback function is a function passed as an argument to another function, to be executed later.

Question 10: What is the purpose of the 'this' keyword in JavaScript?

The 'this' keyword refers to the current object in the context in which it is used in JavaScript.
